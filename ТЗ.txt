1. Цель проекта:
создать приложение на python с интерфейсом pyqt, которое может быть использовано для отвслеживания номеров автомобилей.
Описание приложения:
1 загрузка в приложение фотографии
2 последующая обработка фотографии
3 функционал для работы с БД
4 функционал для настройки нейросети
5 итоговая таблица найденных номеров
2.1 Типы пользователей 
В приложении предусматриваются два типа пользователей, разработчик и обычный пользователь. 
Разработчик может тренировать нейросеть, добавлять новые функции в функционал приложения, осуществлять отладку программы. 
Обычным пользователям предоставляется возможность использовать приложение по назначению.
2.3 Итоговая таблица вошедших людей
Приложение будет создавать таблицу, в которой будет 2 колонки: 
№, номер автомобиля.
3. Предлагаемый стек технологий:
1 Python
2 Библиотеки PyQt, sqlite3, PIL, easyocr, cv2
3 Нейросеть Yolov8
